MyShop - Static E-commerce Demo
==============================

Files:
- index.html
- style.css
- script.js
- images/ (SVG placeholder images)

How to use:
1. Extract the zip.
2. Upload all files to a GitHub repository root (or to Netlify) and deploy.
3. On Netlify: "Add new site" -> "Deploy manually" and drag & drop the folder, or connect to GitHub repo.
4. On GitHub Pages: push the files to a public repository and enable Pages (branch: main, folder: root).

Notes:
- This is a static demo. No real payments or backend.
- You can edit `script.js` to change products or behavior.

Enjoy!
